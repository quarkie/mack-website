Infrastruktur Übersicht

Die zugrundeliegende Plattform basiert auf einem physischen 2-Node Cluster plus Quorum-Device, verwaltet durch striktes Infrastructure as Code (IaC).

  • Node 01: HP t740 (Proxmox VE)
  • Node 02: HP t740 (Proxmox VE)
  • QDevice: Raspberry Pi (Corosync)
  • Storage: Lokales NVMe (TLC NAND)

Deployment Strategie

Manuelle Eingriffe auf der Hardware sind auf die initiale IP-Vergabe beschränkt.

Die gesamte Konfiguration erfolgt deklarativ. Änderungen am Systemstatus werden ausschließlich über Versionskontrolle und CI/CD-Pipelines eingespielt.
